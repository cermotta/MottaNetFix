# MottaNetFix

O **MottaNetFix** é um utilitário técnico desenvolvido para automatizar o diagnóstico e o reparo de problemas de conectividade de rede no Windows. 

## Funcionalidades
* **Reset Completo de Pilha TCP/IP e Cache:** Executa uma sequência de comandos (netsh/ipconfig) para restaurar a integridade da rede.
* **Limpeza de Cache DNS:** Remove dados temporários que impedem o acesso a sites.
* **Renovação de IP:** Solicita um novo endereço IP ao servidor DHCP.
* **Configuração de DNS (Google):** Otimiza a latência e a estabilidade da navegação.
* **Diagnóstico Profissional:** Realiza testes de Ping (Roteador/Google) e mede em tempo real a velocidade de **Download**, **Upload** e **Latência (Ping)** utilizando servidores profissionais.

## Como usar
1. Baixe o executável `MottaNetFix.exe` na seção de Releases.
2. Clique com o botão direito e selecione **"Executar como Administrador"** (necessário para aplicar mudanças no sistema).
3. Siga as opções do menu interativo para o procedimento desejado.

## Requisitos Técnicos
* Windows 10 ou superior.
* Conexão com a internet para as funções de teste de velocidade e DNS.
